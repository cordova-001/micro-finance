## About Micro Finance

It is a special project developed to suit the operations of Microfinance institutions such as:

- Microfinance Banks.
- Esusu Management Operation
- Cooperative Society Management
- Mini-Bank Operations


Laravel is accessible, powerful, and provides tools required for large, robust applications.
